学习笔记

```
本周学习的数据结构是数组，链表，栈，队列以及在此基础上构造出来的跳表，优先队列，双端队列等。比较这些数据结构的特点，可以发现它们都是一种线性的数据结构。除了首尾元素外，每个元素有且仅有一个前驱和后继。在解决具体的问题的时候，具体使用其中的哪一种需要根据它们的特性来做出判断。比如针对查找，增加删除操作，不同的时间复杂度；元素的出入顺序等。要想灵活使用它们，不仅需要对它们的特定有清晰的认识，还要积累足够多的经验，做到见多识广，这样在遇到陌生的题目的时候，能够根据记忆快速判断应该使用哪种数据结构来解决以及在使用的时候运用哪些技巧。有些技巧不是单纯在那里想就能轻易想出来的，因此必须通过“五毒神掌”将它们牢牢掌握，形成一种条件反射般的记忆。当然，如果只是单纯地积累做题经验而不去思考为何要使用这种方法和技巧，遇到灵活性比较大的题目，很容易束手无策。因此最好的学习方法和态度就是既要重视模仿练习，也要进行深入的思考和探索，最终达到真正掌握。
```
